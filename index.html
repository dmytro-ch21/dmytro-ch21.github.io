Checkboxes

Note on Checkboxes: Checkboxes allow for the selection of multiple options. This is commonly seen in forms where users can select all that apply.

Radio Buttons

Note on Radio Buttons: Radio buttons are used when only one option should be selected from a group. Selecting a new option will unselect the previous one.

Single Select Dropdown

Note on Single Select Dropdown: A single select dropdown allows you to choose one option from a list. When you click on the dropdown, all the options become visible. However, only one option can be selected at a time.

Multi-Select Dropdown

Note on Multi-Select Dropdown: You can select multiple options from this dropdown. The box will enlarge as more options are added.
